# 双图编辑功能文档

## 功能概述

双图编辑是一个基于即梦 CLI 的图片处理工具，用户上传两张图片并输入描述，即可生成编辑后的图片/视频。

**使用路径：** `/tools` → 选择"双图编辑" → 点击"立即使用"

## 技术实现

### 目录结构

```
app/
├── api/
│   └── tools/
│       └── image-blend/
│           └── route.ts          # API 端点
└── (app)/
    └── tools/
        └── image-blend/
            └── page.tsx          # 工具页面
foundation/
└── providers/
    └── JimengCliProvider.ts     # 即梦 CLI 封装
```

### API 接口

**端点：** `POST /api/tools/image-blend`

**请求体：**
```json
{
  "imageA": "https://example.com/image-a.jpg",    // 图片A URL (必填)
  "imageB": "https://example.com/image-b.jpg",    // 图片B URL (必填)
  "prompt": "将图片B中的人物融合到图片A的场景中", // 编辑描述 (必填)
  "options": {                                    // 可选参数
    "duration": 5,
    "ratio": "9:16",
    "modelVersion": "seedance2.0",
    "videoResolution": "720p"
  }
}
```

**响应：**
```json
{
  "success": true,
  "outputUrl": "https://example.com/output.jpg"
}
```

### JimengCliProvider 类

核心方法：

| 方法 | 说明 |
|------|------|
| `imageBlend(imageA, imageB, prompt, options)` | 双图编辑，生成混合图片/视频 |
| `image2video(imageUrl, prompt, options)` | 图生视频 |
| `queryResult(submitId, downloadDir?)` | 查询任务结果 |
| `listTasks(status?)` | 列出任务列表 |

## 使用流程

1. **上传图片A** - 点击图片A区域选择文件上传
2. **上传图片B** - 点击图片B区域选择文件上传
3. **输入描述** - 在文本框中输入编辑指令
4. **点击生成** - 调用即梦 CLI 执行图片编辑
5. **查看结果** - 生成完成后显示结果图片

将图片A的艺术风格应用到图片B的人物上
